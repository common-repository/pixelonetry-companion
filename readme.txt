This plugin creates the additional sections on the home page for pixelonetry Themes

== Description ==

This plugin creates the additional sections on the home page for pixelonery themes. By installing the PIXELONETRY Companion plugin.
Pixelonetry Companion is a plugin created to enhance front Page Sections the functionality of the WordPress theme created by pixelonetry.

== Features ==

Slider Section
Info Section
Service Section
Call To Action Section 
Team Section
Pricing Section
About Section
Contact Section
And many more.
